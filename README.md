# Калькулятор

## Описание
Пробный сайт калькулятора с простыми функциями:
- сложение
- вычитание
- умножение
- деление

Также реализованна возможность вводить цифры и операторы с помощью клавиатуры

## Особенности
Сайт адаптирован под планшеты и смартфоны
